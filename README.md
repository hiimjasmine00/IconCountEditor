# Icon Count Editor
Allows you to edit the game's icon counts.

## Features
- Editable icon counts, which can be changed in the mod settings
  - Cube
  - Ship
  - Ball
  - UFO
  - Wave
  - Robot
  - Spider
  - Swing
  - Jetpack
  - Death Effect
  - Trail
  - Ship Fire
